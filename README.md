# nodejs-telegram
A simple Node.js app that send messages to Telegram users.

## How to
1. Enter in the nodejs-telegram folder with cd
2. Create a .env file with the same vars as .env.example
3. Put your data in the .env file
4. Run npm install
5. Run node index.js
